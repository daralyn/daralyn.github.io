"Wobbler" Steam Engine Overview

The Wobbler Steam Engine is a simple steam-engine design that was used in the past as to run ships and other engines. Even though the design is no longer in use on the large scale, it's fascinating to see how it works. The design that we made, inspired by a design on Mr. Pete's YouTube channel, is run with compressed air, instead of steam.
We began on September 6, 2017 by looking at videos on the design and how it works, and then started designing it on OnShape, an online Computer Aided Design (CAD) application. The engine consists of 7 main parts: a wood base, tower, flywheel, flywheel axel, crank wheel, cylinder head, and piston assembly. For this project, we mainly used the the lathe and the mill. I learned how to use both, the applications of both, and I also got more comfortable with the belt sander and grinder.

First G-code Exercise Overview

In order to get more familiar with the machines we're using in Engineering, we learned a little bit about the code that these machines use: G-code. We were assigned to create a line, patterned four times. We changed depths and feed speeds in order to also see how those can affect the quality of a piece. To test our codes, we used an online tool (http://chilipeppr.com/tinyg) to simulate what the tool path would look like.

Flat Pack Project Overview

In this project, we created a 3D object that could be pressed together with multiple 2D pieces. The constraints were that the piece should fit on a 2'x5' board of plywood, and could use either 1/2" or 1/4" thickness. We again used the Maverick Router. My pieces were cut on the 1/4" thick plywood with a 1/8" end mill.

Geneva Mechanism Overview

For this project, we looked at videos showing how the Geneva Mechanism moves/works. Then we had to design one ourselves and think about how it moves and think about clearances between parts. Larger clearances require less design accuracy but also can cause it not to operate correctly. Having no clearances would not allow it to move. We used the laser cutter to cut the pieces out and test if the clearances worked. I made adjustments, and then we used Fusion 360's CAM software to generate code to cut out the drive wheel. We then cut out the remaining pieces on the water jet, and assembled it.
